# 🌑 ECLIPSE

A story-driven 2D survival horror game built entirely with **HTML**, **CSS**, and **JavaScript**.

ECLIPSE follows a group of survivors as they fight to stay alive in a world overrun by the infected. The game combines exploration, environmental storytelling, puzzles, cinematic cutscenes, and emotional character interactions into a chapter-based survival horror experience.

---

## 🎮 Features

- Multi-chapter story campaign
- Original characters and narrative
- Interactive dialogue system
- Puzzle-based progression
- Survival horror gameplay
- Cinematic cutscenes
- Mobile-friendly controls
- Landscape gameplay experience
- Installable Android application
- Progressive Web App (PWA) support

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Capacitor (Android Packaging)
- Progressive Web App (PWA)
- Vercel

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/SadiaaAhmad/ECLIPSE.git
```

Open the project folder and launch a local server.

Example using Python:

```bash
python -m http.server 8000
```

Then open:

```
http://localhost:8000
```

---

## 📱 Android Build

This project can be packaged as a native Android application using Capacitor.

```bash
npm install
npx cap copy android
npx cap open android
```
